# bboss rpc工程
 bboss rpc eclipse project.支持丰富的协议栈(http/netty/mina/jms/webservice/rmi/jgroups/restful) 安全高效,
 
   可非常方便地将bboss ioc管理的业务组件发布成RPC服务

# 工程构建运行说明：
## 1.搭建好ant构建环境和jdk 1.7及以上
## 2.运行工程根目录下的build.bat指令
## 3.构建成功后,会生成两个文件：

distrib/bboss-rpc.jar
